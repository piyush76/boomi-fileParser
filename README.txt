Boomi File parsing assignment
--------------------------------------------------------------------------------

1. Run mvn clean install

2. This will generate the boomi-fileupload.war under target directory

3. deploy the boomi-fileupload.war into the tomcat/webapps dirctory.

4. Open the browser and request the URL http://127.0.0.1:8080/boomi-fileupload/uploadFile.do

5. Upload the file boomi.txt for testing.


File validations are in place for empty file, .txt file.
